# Evidence Labels

Use these labels on important claims:

| Label | Meaning | Allowed use |
|---|---|---|
| `[来自文献]` | Directly supported by uploaded/read papers | Literature cards, evidence tables, findings, limitations |
| `[来自用户材料]` | Directly from user notes, data, files, or stated goals | User research direction, dataset description, target journal |
| `[方法推断]` | Reasonable methodological inference from field knowledge | Variable design, model route, data preprocessing |
| `[待验证假设]` | Plausible but not yet supported | Innovation claims, expected results, mechanisms |
| `[写作建议]` | Rhetorical or structural advice | Titles, paragraph logic, sentence templates |

Rules:

1. Do not mix direct evidence and speculation in one sentence without labels.
2. Mark missing evidence as a gap instead of filling it with invented detail.
3. For citations, require a real source path, DOI, database record, or user-provided paper before treating it as formal evidence.
