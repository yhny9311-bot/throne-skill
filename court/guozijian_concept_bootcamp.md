# 国子监: Concept Bootcamp

Use for `入学：概念启蒙`.

## Output Format

For each concept:

1. 这是什么: explain in plain Chinese.
2. 它解决什么问题: the research problem it helps answer.
3. 风景园林怎么用: concrete landscape/urban ecology application.
4. 最小方法链: data -> indicator -> analysis -> interpretation.
5. 常见误区: what not to overclaim.
6. 入门阅读方向: give search terms or user-provided papers; do not invent citations.
7. 与用户课题的关系: tailored relevance.

Use `references/concept_glossary.md` first, then reason from the user's material.
